# SIG_Prompts_GEODRAFT
GEODRAFT: 
G : Groupe cible (profil SIG)
E : Environnement SIG
O : Objectif métier / analytique
D : Données (structure & qualité)
R : Rôle de l’IA
A : Action demandée
F : Format de sortie
T : Tonalité & niveau technique
Cette structure reste cohérente avec les frameworks de prompt engineering existants, tout en parlant le langage des métiers SIG un cadre structuré de conception de prompts permettant de définir précisément l’audience, la posture de l’IA, le résultat attendu, l’action à produire, les contraintes contextuelles, le format du livrable et la tonalité, afin d’obtenir des réponses fiables, cohérentes et orientées impact.
